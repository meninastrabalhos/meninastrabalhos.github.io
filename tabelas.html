Tabelas HTML

Tabelas

As tabelas em HTML permitem que os desenvolvedores organizem os dados em linhas e colunas. A estrutura da tabela é feita entre as tags "< table >" e "< /table >". 

           <!DOCTYPE>
           <html> 
            <table>
                ..... 
                .....         
            </table>
           </html>


linhas e colunas

As linhas da tabela começam com "< tr >" e terminam com "< /tr >". Mas utilizando somente a tag, ela não irá funcionar sozinha. Então é utilizado a tag "< td >" e "< /tr >" para definir as células da tabela que é onde fica o conteúdo. Pode-se ter quantas linhas for preciso, contanto que a quantidade de células tenha a mesma quantidade em cada linha

Exemplo:

            <!DOCTYPE html>
            <html>  
              <table>
               <tr>
                <td> abc </td>
                <td> def </td>               
               <tr>
               

               <tr>
                <td> 123 </td>
                <td> 456 </td>               
              <tr>          
              </table>
            </html>

Dessa forma terá duas células.


abc def
123 456

Cabeçalho

Para cabeçalhos na tabela utiliza-se o "< th >" abrir e "< /th >"para fechar em vez de "< td >" e "< /td >". Pois desta forma os elementos da célula ficam em negrito e centralizados. Porém podem ser modificados com CSS.

Exemplo:

                <!DOCTYPE html >
                <html>
                 <table>
                  <tr>
                    <th> abc </th>
                    <th> def </th>
                  </tr>
        
                  <tr>
                    <td> 123 </td>
                    <td> 456 </td>
                  < /tr>         
                 </table>
                </html>

abc def
123 456

Borda

As tabelas html podem conter bordas de diferentes estilos.

Ao adicionar bordas em uma tabela, você adiciona bordas em volta de cada célula. Para adiciona-lá é preciso adicionar o atributo "border" propriedade CSS nos elementos "table", "th" e "td"

            <!DOCTYPE html >
                <html>
                 <head>
                    <style>
                        table, th, td {
                          border: 2px solid rgb(192, 107, 181);
                        }
                    </style>
                 </head>

                 <table>
                  <tr>
                    <th> abc </th>
                    <th> def </th>
                  </tr>
        
                  <tr>
                    <td> 123 </td>
                    <td> 456 </td>
                  < /tr>         
                 </table>
                </html>


abc def
123 456


Borda recolhida

No HTML, para retirarmos as bordas duplas (como no exemplo acima), devemos definir a "border-collapse", que é uma propriedade CSS como collapse.

Exemplo de como fazer a tabela ficará com esse processo:

            <!DOCTYPE html >
                <html>
                 <head>
                    <style>
                        table, th, td {
                          border: 2px solid rgb(192, 107, 181);
                          border-collapse: collapse;
                        }
                    </style>
                 </head>

                 <table>
                  <tr>
                    <th> abc </th>
                    <th> def </th>
                  </tr>
        
                  <tr>
                    <td> 123 </td>
                    <td> 456 </td>
                  < /tr>         
                 </table>
                </html>


abc def
123 456


Preenchimento

O preenchimento da célula é o espaço entre o conteúdo e as bordas da célula. O padrão definido é 0 pixel. Para adicionar o preenchimento de células, utilize o "padding" propriedade CSS. 

                <!DOCTYPE html >
                    <html>
                     <head>
                        <style>
                            table, th, td {
                              border: 2px solid rgb(192, 107, 181);
                            }
                            
                                th, td{
                                padding: 30px;
                            }
                            
                        </style>
                     </head>
    
                     <table>
                      <tr>
                        <th> abc </th>
                        <th> def </th>
                      </tr>
            
                      <tr>
                        <td> 123 </td>
                        <td> 456 </td>
                      < /tr>         
                     </table>
                    </html>
abc def
123 456

Para adicionar preenchimento somente acima do conteúdo, use a "padding-top" propriedade.

E os outros lados com as propriedades " padding-bottom ", " padding-left " e " padding-right"

Exemplo de como fazer a tabela ficar com esse processo:

th, td {
padding-top: 10px;
padding-bottom: 20px;
padding-left: 30px;
padding-right: 40px;
}



Espaçamento

O espaçamento das células é o espaço entre as telas. o padrão definido é de 2 pixels. Para alterar esse espaço, utilize "border-spacing" propriedade CSS no "table" elemento.

                <!DOCTYPE html >
                    <html>
                     <head>
                        <style>
                            table, th, td {
                              border: 2px solid rgb(192, 107, 181);
                            }
                            
                            table {
                                border-spacing: 30px;
                                }
                            
                        </style>
                     </head>
    
                     <table>
                      <tr>
                        <th> abc </th>
                        <th> def </th>
                      </tr>
            
                      <tr>
                        <td> 123 </td>
                        <td> 456 </td>
                      < /tr>         
                     </table>
                    </html>
abc def
123 456
