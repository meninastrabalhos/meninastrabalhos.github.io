Introdução ao HTML

O que é HTML?

HTML significa Hyper Text Markup Language e é a linguagem de marcação padrão para criar páginas da Web

Ele descreve a estrutura de uma página da Web e consiste em uma série de elementos

Os elementos HTML informam ao navegador como exibir o conteúdo e rotulam partes do conteúdo dizendo onde é um parágrafo, um link, uma imagem..., etc.

Primeira página de html

Para fazer a primeira página de html, digite o código abaixo no bloco de notas ou Visual Studio Code:

        <!DOCTYPE html>
        <html>
          <body>
            Essa é a minha primeira página em html!
          </body>
        </html>

Salve o arquivo como index.html e abra-o em algum navegador

Exemplo explicativo em HTML 

 <!DOCTYPE html> É a declaração define que este documento é um documento HTML5 

 <html> É  o elemento é o elemento raiz de uma página HTML 

 <head> É o elemento contém metainformações sobre a página HTML 

 <title> É  o elemento especifica um título para a página HTML (que é mostrado na barra de título do navegador ou na guia da página) 

 <body> É o elemento define o corpo do documento e é um recipiente para todos os conteúdos visíveis,
            como cabeçalhos, parágrafos, imagens, hiperlinks, tabelas, listas, etc. 

 <h1> É o elemento define um título grande 

 <p> É  o elemento define um parágrafo

O que é um elemento HTML ?

Um elemento HTML é definido da seguinte maneira: uma tag inicial, algum conteduo após ela e uma teg final 

 <tagname> O conteúdo fica posicionado aqui </tagname>

Desta forma, o conteúdo HTML é tudo, desde a tag inicial até a tag final 

 <h1> Esta é minha primeira pagina HTML </h1> 

 <h1> Este é meu primeiro cabeçalho em HTML </h1> 

 <p> Este é meu primeiro parágrafo em  HTML </p>
