Box model CSS

Box Model(Modelo de Caixa)

O modelo de caixa CSS (Box Model) é essencialmente uma caixa que envolve cada elemento HTML.

Ele possui: margens, bordas, preenchimento e o conteúdo real. A imagem abaixo ilustra esse modelo:

  1. Conteúdo(Content): é onde aparecem textos e imagens.
  2. Preenchimento(Padding): Limpa uma área ao redor do conteúdo. O padding é transparente.
  3. Borda(Border): ela contorna o preenchimento e o conteúdo.
  4. Margem(Margin): Limpa uma área fora da borda. A margem é transparente.

O modelo de caixa nos permite adicionar uma borda ao redor dos elementos e definir o espaço entre eles.

Exemplo usando box model:

     <!DOCTYPE html>
     <html>
         <head>
             <style>
                 p{
                     width: 320px;
                     padding: 10px;
                     border: 5px solid orange;
                     margin: 5px;
                     background-color: pink;
                 }
             </style>
         </head>  
         <body>
         <p>Conteúdo do Box Model</p>    
         </body>
     </html>
