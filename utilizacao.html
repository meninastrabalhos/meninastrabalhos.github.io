Formas de utilização do CSS

Existem três maneiras de inserir o CSS:

  1. Externo
  2. Interno
  3. Embutido

CSS Externo

Para inserir o CSS de maneira externa, é necessário criar um arquivo que seja "index.css"

Na página HTML deve-se referenciar esse arquivo dentro do elemento "link" e dentro da seção "head"

Exemplo da página HTML:

	<!DOCTYPE html>
        <html>
            <head>
                <link rel="stylesheet" href="mystyle.css">  ->local onde deve referenciar o arquivo "index.css"
            </head>  
            <body>
            </body>
        </html>

Na folha de estilo externa não pode ter tags HTML

Nela é possível alterar a aparência de um site inteiro alterando apenas seu arquivo

Exemplo:

            body{
                background-color: red;
            }

            p{
                color: blue;
            }

CSS Interno

O estilo interno é definido dentro do elemento "style", dentro da seção head

Veja o exemplo a seguir:

                <!DOCTYPE html>
                    <html>
                        <head>
                            <style>
                                body{
                                    background-color: red;
                                }
                      
                                p{
                                color: blue;
                                margin-left: 35px;
                                }
                            </style>
                        </head>  
                        <body>
                            <p>Isto é um parágrafo</p>
                        </body>
                    </html>

CSS Embutido

Um estilo embutido pode ser usado para aplicar um estilo único a um único elemento.

Para usar estilos embutidos, adicione o atributo style ao elemento relevante. O atributo style pode conter qualquer propriedade CSS.

Exemplo de CSS embutido:

            <!DOCTYPE html>
            <html>
                    <head>
                    </head>  
                    <body>
                        <h1 style="color:green;text-align:center;">Isto é um cabeçalho</h1>
                        <p style="color:black;">Isto é um parágrafo</p>
                    </body>
            </html>
