Tags em HTML

O que é uma tag?

As tags são marcadores usados para delimitar secões de um documento HTML. Podemos fazer a sintaxe de uma tag HTML utilizando os símbolos: <>

Além disso, outro ponto importante para destacar é que em alguns casos nós devemos declarar as tags em pares, ou seja, ao declará-las devemos abrir e fechar o escopo.

Exemplo: Ao declararmos a tag de paragráfo, escrevemos da seguinte maneira: 

        <p> </p>

Como iniciar uma página em HTML?

Ao iniciar uma página em HTML é importante que lembremos de inserir a declaração: 

        <!DOCTYPE html>

Mas afinal, por que isso é importante? É importante para que o navegador consiga identificar o tipo de documento que ele irá interpretar.

Além disso, outro ponto importante é que o conteúdo da página deve estar entre: 

        <html> </html>

Como fazer um cabeçalho em HTML?

Para fazer um cabeçalho em HTML utilizamos as tags:

        <head> </head>

O cabeçalho é um espaço que conterá as informações iniciais da página, por exemplo, título e sub título.


Como delimitar o escopo da página?

O escopo da página (corpo) pode ser delimitado por meio da tag:

        <body> </body>

Esse elemento define o corpo do documento e é um recipiente para todos os conteúdos visíveis, como parágrafos, imagens, vídeos, hiperlinks, tabelas, listas, etc.


Como fazer comentários em HTML?

Em HTML os comentários são feitos usando: 

        <!--Comentário-->

Como quebrar uma linha em HTML?

A quebra de linha, também conhecida como retorno suave, possui a mesma funcionalidade do recurso "enter", ou seja, serve para começar uma nova linha sem iniciar um novo parágrafo.

        <br>

Como fazer um parágrafo em HTML?

A tag utilizada para fazermos um parágrafo é:

        <p> </p>

Como colocar o meu texto em estilo de cabeçalho (título)?

Há 6 diferentes tamanhos que podem adicionados, quanto menor o número, maior a fonte.

Por exemplo: A maior fonte disponível é adicionado por meio da tag h1, a menor tag disponível é adicionada por meio da tag h6.

A tag utilizada é:

        <h2> </h2>

Como inserir imagens na página?

Para melhorar a estética do site ou para fins informativos, podemos adicionar imagens na página e isso é feito por meio da tag:

A tag utilizada é:

        <img src="arquivo-imagem">

Como criar hiperlinks na página?

Para que servem o link? Servem para que ao clicar nele, o usuário seja encaminhado para outra página. Em HTML links são hiperlinks.

A tag utilizada é:

        <a href="link"> Texto </a>

Como inserir uma pequena imagem na guia da página?

As imagens inseridas podem ser de autoria própria ou baixadas gratuitamente em algum outro site.

A tag utilizada é:

        <link rel="icon" href="image-file" type="image/x-icon">
